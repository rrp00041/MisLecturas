# Roadmap de Implementación

## Fase 1 — Base técnica
- Crear proyecto Flutter multiplataforma.
- Integrar Firebase Core, Auth, Firestore y Storage.
- Configurar entornos (`dev`, `prod`) y secretos.
- Definir reglas de seguridad Firestore.

## Fase 2 — Biblioteca y búsqueda
- Pantalla login/logout.
- CRUD de libros manual.
- Integración Google Books (título/autor/ISBN).
- Importación con edición de campos personalizados.

## Fase 3 — Organización
- Estados de lectura predefinidos.
- Creación de listas dinámicas ilimitadas.
- Filtros por estado/lista/tags/rating.

## Fase 4 — Dashboard analítico
- Barras: libros por mes/año.
- Línea: páginas por día/semana.
- Donut: géneros y autores.
- Heatmap de lectura por día de semana.
- Predicción ETA de finalización libro actual.

## Fase 5 — Pulido y entrega
- Modo oscuro + accesibilidad.
- Optimización de consultas e índices.
- QA funcional multiplataforma.
- Build APK firmada + build Web/Desktop.

## Criterios de aceptación
- Login funcional y datos aislados por usuario.
- Sincronización móvil/escritorio en tiempo real.
- Importación desde Google Books en 1 clic.
- Dashboard con las 5 visualizaciones solicitadas.
