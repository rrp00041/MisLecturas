# Roadmap

## Hecho

- Proyecto Flutter multiplataforma inicializado.
- Persistencia local migrada a Isar.
- Búsqueda en Google Books.
- Biblioteca con diseño premium, `Hero` y animaciones.
- Ficha de lectura con slider de progreso, etiquetas y cronología.
- Dashboard con datos reales desde la base local.

## Siguiente iteración

- Crear colecciones personalizadas además de etiquetas.
- Añadir filtros avanzados por tag, género y rating.
- Incorporar exportación/importación de backups locales.
- Mejorar búsqueda offline con caché local.
- Añadir tests de integración para flujos completos.
